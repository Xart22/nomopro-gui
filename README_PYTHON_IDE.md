# 🐍 Panduan Python IDE - Nomopro

## 📖 Panduan Lengkap Penggunaan Python IDE

Panduan ini dirancang untuk membantu **siswa SD** dan **mahasiswa** menggunakan Python IDE dengan mudah dan efektif.

### 📚 Isi Panduan

1. **Pengenalan Python IDE** - Memahami 3 mode utama: Upload, Realtime, dan REPL
2. **Mode Upload** - Cara flash firmware MicroPython ke board
3. **Mode Realtime** - Testing kode cepat tanpa hardware
4. **Mode REPL** - Debugging dan testing interaktif
5. **Built-in Functions** - Referensi lengkap 100+ fungsi yang tersedia
6. **Tips & Tricks** - Contoh kode dan best practices
7. **Troubleshooting** - Solusi masalah umum untuk SD dan Mahasiswa
8. **Quick Reference Card** - Ringkasan yang bisa dicetak

### 🎯 Fitur Utama

- ✅ **100+ Built-in Functions** - Sudah terintegrasi, tidak perlu install
- ✅ **3 Mode Programming** - Upload, Realtime, dan REPL
- ✅ **Multi-Device Support** - Arduino, ESP32, Micro:bit, Raspberry Pi Pico, NomoBot
- ✅ **Visual Editor** - Monaco editor dengan auto-completion
- ✅ **Real-time Debugging** - Serial terminal dan REPL

### 🚀 Cara Memulai

1. **Baca Panduan Lengkap** di `PANDUAN_PYTHON_IDE.md`
2. **Mulai dari Realtime Mode** untuk belajar dasar Python
3. **Flash MicroPython** ke board menggunakan Upload Mode
4. **Test di REPL** untuk debugging
5. **Upload & Run** untuk menjalankan di hardware
